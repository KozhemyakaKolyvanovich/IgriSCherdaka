Платформа для публикации пользовательских и редакционных оценок инди-игр. Пользователи могут ставить оценки, писать рецензии, просматривать рейтинги и фильтровать игры по жанрам, дате выхода или средней оценке. Реализовано на WordPress с кастомной логикой рейтингования.
 Цели проекта

- Дать инди-геймерам и разработчикам единую площадку для честных оценок и обзоров.
- Внедрить систему взвешенного рейтинга (оценка + полезность отзыва).
- Обеспечить удобную навигацию по играм, жанрам и оценкам
